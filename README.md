# ShareSafe API
A SpringBoot API to securely share files with others. 

The following list of REST services provide, users to securely upload files to AWS S3 storage and share with others.

## List of services 
* Files Services
  * Create File
  * Upload File
  * Delete File
  * List all files


* User Services
